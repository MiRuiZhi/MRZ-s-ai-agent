# CSDN 博客格式指南

## 标题规范

- 5-100 字符
- 包含关键词便于搜索
- 格式：`{主题}：{副标题/描述}`
- 示例：`AI Agent 的崛起：从概念到实践的全面解析`

## 文章结构

```markdown
# 标题

[开篇引言，1-2 段，说明文章背景和价值]

## 📑 文章目录

- [一. 第一章 emoji](#一-第一章-emoji)
  - [1.1 小节](#11-小节)
  - [1.2 小节](#12-小节)
- [二. 第二章 emoji](#二-第二章-emoji)
...
- [参考文献](#参考文献)

---

## 一. 第一章标题 emoji

[正文内容]

### 1.1 小节标题

[详细内容]

思考：💡 [引发读者思考的问题]

🤔 [问题的解答或讨论]

---

## 参考文献

[1] Author. "Title." Source. URL
[2] ...
```

## Emoji 使用规范

| 章节类型 | 推荐 Emoji |
|---------|-----------|
| 起源/背景 | 🌅 |
| 架构/结构 | 🏗️ |
| 能力/特性 | 🎯 |
| 协作/团队 | 👥 |
| 工具/框架 | 🛠️ |
| 案例/示例 | 🌟 |
| 未来/展望 | 🚀 |
| 挑战/问题 | ⚠️ |
| 总结 | 📝 |

## 表格格式

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|

```

## 思考问题格式

```markdown
思考：💡 [问题]

🤔 [解答，通常 2-4 句话]
```

## 引用格式

```markdown
> 💡 **重点提示**
>
> 引用内容...

**案例**：[案例描述]
```

## 参考文献格式

```markdown
[1] 作者. (年份). "标题." 来源. URL

[2] GitHub项目. URL
```

## 字数建议

| 文章类型 | 字数范围 |
|---------|---------|
| 快速教程 | 3,000-5,000 |
| 技术分享 | 5,000-10,000 |
| 深度综述 | 10,000-20,000 |
| 系列文章 | 每篇 5,000-8,000 |

## 发布设置

| 设置项 | 推荐值 |
|-------|-------|
| 文章类型 | 原创 |
| 可见范围 | 全部可见 |
| 摘要 | AI 提取或手写（256字内）|
| 标签 | 2-5 个相关标签 |
| 封面 | 16:9 比例，与主题相关 |
